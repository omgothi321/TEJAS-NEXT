# Tejas — AI Operating System

> Controls your computer locally. No cloud. No subscriptions. Your data never leaves your machine.

## See it in action

[Demo Video — coming soon]

## What Tejas does

```bash
tejas "summarize all PDFs in Downloads and send to Telegram"
```

✓ Finds your PDFs  
✓ Reads and summarizes them  
✓ Sends report to your Telegram  
✓ Speaks the result aloud  

All on your machine. Zero cloud.

## Install

```bash
git clone https://github.com/omgothi321/tejas-next
cd tejas-next
npm install
node bin/tejas.js setup
```

## Why not Claude Code or ChatGPT?

They run on someone else's servers.  
Tejas runs on yours.  
Your files, your memory, your control.

## Works on

Linux · macOS · Windows (coming soon) · Raspberry Pi

## Built with

- Local AI models via Groq / Gemini / Ollama
- SQLite memory that compounds over time  
- Voice input + output (offline)
- Telegram remote control
- 344 specialized AI agents
